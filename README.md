# TP Traducteur Français - Anglais

## Participants
- Zougari Zineddine
- Wassim Bensefia
- Ali Kaddour-Djebbar

## Description
Ce projet est un mini-site web PHP qui traduit des mots du français vers l'anglais et inversement.

## Fonctionnalités
- Traduction FR → EN et EN → FR
- Normalisation des mots (minuscules, suppression des espaces)
- Gestion des mots inconnus avec liste des mots connus
- Affichage du dictionnaire complet
- Gestion des erreurs (mot vide ou sens non choisi)
